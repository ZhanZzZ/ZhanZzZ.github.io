Vue组件之间通讯方式

Props / $emit（父子组件通信）

Props（父传子）
$emit（子传父）

v-model（双向绑定）

Provide / Inject（跨层级通信）

EventBus（事件总线）

Vue 3中推荐使用第三方库如mitt：

Pinia/Vuex（状态管理）

Pinia（Vue 3推荐）

attrs/listeners（属性透传）

ref / expose（父组件直接访问子组件）

parent/root（不推荐）

选择建议

  • 父子通信：优先使用Props / $emit
  • 双向绑定：使用 v-model
  • 跨多层组件：使用 Provide / Inject
  • 兄弟组件：使用 EventBus 或状态管理
  • 全局状态：使用 Pinia/Vuex
  • 复杂应用：推荐 Pinia 进行状态管理

Vue 2 vs Vue 3 差异

Vue 3的变化：

  • 移除了 $listeners（合并到 $attrs）
  • 移除了 EventBus 的官方支持（需使用第三方库）
  • 推荐使用 Pinia 而不是 Vuex
  • Composition API 使得逻辑复用更简单
